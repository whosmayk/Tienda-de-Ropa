# Tienda-de-Ropa
CRUD para Proyecto de Tienda de ropa para Taller de Bases de Datos


La carpeta "CRUD Tienda Ropa" contiende la carpeta "tienda_ropa", la cual contiene los archivos .php para phpmyAdmin

También incluye el dump de la base de datos y poder cargarla en phpmyAdmin

La carpeta Scripts contiene los scripts utilizados para demostración en presentación del proyecto.
